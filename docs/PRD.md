# Prompt to Canvas

AI 增强的可视化图表生成 Skill —— 让你像资深专家一样呈现内容。

## 核心能力

| 能力 | 说明 |
|------|------|
| **AI 专家分析** | 作为10年+经验专家，深度分析你的内容结构 |
| **智能可视化** | 自动推荐最优图表类型和布局策略 |
| **即时编辑** | Excalidraw 驱动的专业级编辑体验 |
| **一键导出** | PNG / SVG 多格式导出 |

## 使用场景

- **Portfolio 展示**：求职面试的个人项目展示
- **述职复盘**：年度/季度工作成果可视化
- **技术分享**：架构图、流程图设计
- **方案汇报**：商业方案、数据看板

## 快速开始

### Step 1：描述你的内容

告诉 AI 你的经历、内容或数据，以及用途。

**示例 Prompt：**
```
帮我把这段经历制作成可视化图表，用于求职 Portfolio：
"关务 AI 数字员工：从 0 到 1 主导关务 AI 数字员工产品设计，
面向中国出口清关跟单场景（23 个口岸，日均 50 至 80 万单），
将运营人工异常处置流程转化为 Agent 自动化执行..."
```

### Step 2：AI 专家分析

AI 作为资深产品经理/专家，深度分析你的内容：
- 识别核心要素（项目、成果、挑战、方案）
- 提取可量化指标
- 设计叙事结构和可视化策略

### Step 3：生成并编辑

系统生成图表，自动打开 Excalidraw 编辑器，你可以：
- 调整文字、颜色、布局
- 添加/删除元素
- 导出为图片
