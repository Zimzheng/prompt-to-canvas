# Prompt to Canvas 用户指南

## 快速开始

### 方式一：Claude Code 中使用

```
/prompt-to-canvas
```

然后描述你的内容和用途即可。

### 方式二：直接描述需求

在任何 Claude Code 对话中，直接说：

```
帮我把这段经历制作成可视化图表：[粘贴你的内容]
```

## 使用示例

### 示例 1：求职 Portfolio

**输入：**
```
帮我把这段经历制作成可视化图表，用于求职 Portfolio：
"从 0 到 1 主导关务 AI 数字员工产品设计，面向中国出口清关跟单场景
（23 个口岸，日均 50 至 80 万单），将运营人工异常处置流程转化为 
Agent 自动化执行。项目成果：任务自动化率 90%，运营提效 50%，
入选 ATA 2026 阿里&蚂蚁集团年度优秀技术项目。"
```

**AI 专家会：**
1. 分析你的经历，识别核心要素
2. 提取量化指标（90%、50%、ATA 2026）
3. 推荐 Portfolio 双栏图布局
4. 生成可编辑的图表

### 示例 2：述职复盘

**输入：**
```
帮我把今年的工作成果制作成可视化图表，用于年度述职
```

### 示例 3：技术架构展示

**输入：**
```
帮我把项目的技术架构制作成架构图
```

## 模板类型

| 模板 | 适用场景 | 布局 |
|------|---------|------|
| **Portfolio 双栏** | 求职/展示 | 左架构，右数据 |
| **时间线** | 职业历程/项目演进 | 纵向时间流 |
| **对比图** | Before/After 效果 | 左右对照 |
| **架构图** | 技术方案/系统设计 | 分层结构 |

## 导出格式

生成图表后，你可以导出为：

| 格式 | 用途 | 说明 |
|------|------|------|
| **PNG** | 常规使用 | 位图，适合嵌入文档 |
| **SVG** | 矢量图 | 可无损缩放，适合设计稿 |

PDF 导出是后续能力；当前版本优先保证 PNG 和 SVG 稳定可用。

## 常见问题

### Q: 需要安装什么吗？
A: 使用前需要执行 `npm install` 和 `npm run build` 生成静态编辑器；Skill 加载后，编辑器会在浏览器中打开。

### Q: 我的内容会被保存吗？
A: 不会。所有数据都在本地和你的会话中，不会上传到任何服务器。

### Q: 可以编辑生成后的图吗？
A: 可以。Excalidraw 编辑器支持文字、形状、手绘、直线和箭头等原生编辑能力。

### Q: 支持团队协作吗？
A: 当前版本是单用户模式。Excalidraw 本身支持协作，但 Skill 层面暂未集成。
